Como instalar?
--------------

Executar `make`.

Como usar?
----------

Primeiro, instale o `snippet`

Digite o começo do `snippet`, ex: "ConTeXt-" ou "mussum" e tecle tab, escolha o `snippet`, tecle <enter>.
	
![](snippets.png)

Outra opção é teclar CTLR-P e gigitar `snippet` ou o nome de um `snippet`, ex: "mussum".
